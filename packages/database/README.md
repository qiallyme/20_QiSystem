# QiSystem Database Adapter (`@qi/database`)

This package provides the core PostgreSQL/Supabase database client wrapper, pooling configurations, and type definitions for **QiSystem** applications.

## Features

* **Database Client Wrapper**: Custom client wrappers wrapping Supabase JS/Postgres connection pooling.
* **Type Definitions**: TypeScript models reflecting the `qisystem` active schemas (`libraries`, `records`, `events`, etc.).
* **Bootstrap Systems**: Commands and procedures for establishing tenant connections and initializing database seeds.
