# QiNode Integration Package (`@qi/qinode`)

This package houses the **QiNode Engine**, which manages local data synchronizations, Markdown-based knowledge base vault interfaces (including Obsidian engine mappings), and local filesystem catalog trackers.

## Features

* **Genesis Root Tracker**: Interfaces for local file trees and asset mappings.
* **Obsidian Sync Engine**: Code for synchronizing structured markdown templates, tags, and layouts.
* **Local Ingest Pipelines**: Watcher integrations that queue local documents into the ingestion database pipelines.
