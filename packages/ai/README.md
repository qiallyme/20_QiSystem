# QiSystem AI Integration Package (`@qi/ai`)

This package provides internal AI capabilities, prompt engineering infrastructure, text classifications, and semantic embeddings support for **QiSystem**.

## Features

* **Ingestion Classification**: Automated parsing and classification of raw imports (e.g. Firefly, Chase transactions) into records.
* **Embeddings Pipelines**: Generating vector representation of documents, notes, and records for local semantic search.
* **Agent Operations**: Tools, structures, and systems coordinating with the `@qi/gina` agent runtime.
